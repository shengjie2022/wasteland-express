# 废土快递 · 游戏策划案 v3.0
## Wasteland Express · Game Design Document

> 角色扮演 / 策略经营 / 废土探索 / 单机游戏
> 版本：v3.0（世界观整合版——派系系统 + 角色系统 + 货物时效 + 零点协议 + 完整世界）
> 制作：盛洁

---

## 一、项目愿景

### 1.1 一句话定位
驾驶改装货车穿越废土，在派系博弈中寻找真相，在贸易与生存中书写自己的传奇。

### 1.2 差异化标签
废土公路旅行 × 派系政治经营 × 叙事驱动探索 × 货物交易策略

### 1.3 核心体验
玩家扮演一名独立货车司机，穿梭于废土走廊的城镇与路线之间。表面上，你在完成货运订单、交易货物、维护车辆——但在这层日常之下，一场关于旧世界真相的谜题正在徐徐展开。派系争夺资源，城镇有自己的故事，而你，作为一个自由人，可以选择站在哪一边，或者保持独立，直到你找到属于自己的答案。

### 1.4 目标用户
- 喜欢策略经营和资源管理的玩家（货物贸易、价格波动、派系关系）
- 享受叙事驱动探索的玩家（废土世界观、旧世界谜题、角色故事）
- 喜爱废土/后末日题材的玩家（行尸走肉、疯狂麦克斯风格）

## 二、世界设定

### 2.1 时间线

| 年份 | 事件 |
|------|------|
| 2077年11月14日 | **大寂静（The Silence）**——AI自动防御系统故障，引发全球核交换。72小时内，9847枚核弹被引爆。 |
| 2077–2087年 | **灰季**——核冬天降临，气温骤降，农作物绝收。全球人口从82亿降至约2亿。 |
| 2087~2117年 | **重建期**——幸存者聚居地形成，派系萌芽。「旧世界货币」因其不可伪造性成为通用货币——瓶盖。 |
| 第87年（游戏时间） | **现在**——新的秩序正在形成，零点协议浮出水面，玩家登场。 |

### 2.2 地理：废土走廊

游戏地图设定在美国西南部废土走廊，横跨亚利桑那州、新墨西哥州、内华达州南部。地图由程序生成，包含以下地形：

| 地形 | 特点 | 策略意义 |
|------|------|----------|
| 熔变荒原 | 高辐射，无定居点 | 高风险高回报的探索区域 |
| 锈蚀平原 | 旧工业区，零件丰富 | 拾荒者联盟势力范围 |
| 骨骸盆地 | 战前高速公路遗迹 | 旧世界遗物出土地 |
| 裂缝峡谷 | 隐秘走私路线 | 锈轮商会控制区 |
| 盐碱滩 | 无遮蔽，无资源 | 仅作为路线穿行区 |
| 再生绿洲 | 农业区，食物充足 | 净土农联核心区 |

**地图边界**：
- 北：灰脊山脉（辐射云笼罩）
- 南：裂痕（地质裂缝，未知）
- 东：锈海（百万废弃车辆堆积）
- 西：零海岸（辐射海滨）

### 2.3 派系六强

#### 拾荒者联盟（Salvage Union）
- **势力色**：橙黄
- **地盘**：锈蚀平原
- **特点**：机械崇拜，低辐射免疫体质
- **立场**：中立偏友好
- **核心诉求**：收集旧世界技术，对抗灰烬之子的「变异者清除」教义
- **与玩家关系**：如果玩家经常进行长途探索，拾荒者是天然的盟友

#### 灰烬之子（Children of the Ash）
- **势力色**：暗灰+骨白
- **地盘**：骨骸盆地北部
- **特点**：宗教组织，医疗水平最高，有极端净化派分支
- **立场**：复杂（视玩家身份而定）
- **核心诉求**：维护废土秩序，对「变异者」持敌意
- **与玩家关系**：如果玩家获得辐射免疫迹象，灰烬之子会变得敌对

#### 锈轮商会（Rustwheel Syndicate）
- **势力色**：深红
- **地盘**：裂缝峡谷走私路线
- **特点**：情报网络强大，商业中立，但涉及灰色交易
- **立场**：绝对中立——只认钱
- **核心诉求**：保持贸易路线畅通，任何破坏稳定的行为都是敌人
- **与玩家关系**：完成走私任务可获得「锈轮通行证」，享受VIP待遇

#### 钢铁脊梁（The Iron Spine）
- **势力色**：军绿
- **地盘**：熔变荒原边缘、旧军事基地
- **特点**：军事化管理，有旧世界武器储备，维持路线秩序
- **立场**：秩序优先，必要时强硬
- **核心诉求**：恢复旧世界秩序，压制派系冲突
- **与玩家关系**：有武器装备会获得尊重，拒绝服从会被驱逐

#### 净土农联（Pure Earth Collective）
- **势力色**：棕+绿
- **地盘**：再生绿洲
- **特点**：粮食生产者，废土上最重要的中立力量
- **立场**：友好中立，警惕任何威胁食物供应链的行为
- **核心诉求**：保护粮食生产，不参与政治
- **与玩家关系**：食物补给首选来源，完成护送任务可获得友谊和折扣

#### 零点追寻者（Zero Seekers）
- **势力色**：黑+荧光绿
- **地盘**：无固定领土，游牧
- **特点**：追寻大寂静真相，寻找零点设施入口
- **立场**：超然——不参与派系政治，只关心真相
- **核心诉求**：找到零点协议的真相
- **与玩家关系**：完成探索任务可获得「零点碎片」，收集全部碎片可解锁真结局

## 四、核心系统详解

### 4.1 派系系统（核心新增）

#### 4.1.1 声望等级

每个派系有5个声望等级：

| 等级 | 名称 | 颜色 | 效果 |
|------|------|------|------|
| -2 | 敌对 | 红 | 商店价格+30%，可能被拦截，禁止进入 |
| -1 | 不信任 | 橙 | 商店价格+15%，任务报酬降低 |
| 0 | 中立 | 灰 | 无特殊效果 |
| +1 | 友善 | 浅绿 | 商店价格-10%，优先接到派系任务 |
| +2 | 信赖 | 深绿 | 商店价格-20%，解锁派系专属装备商店，获得派系通行证 |

#### 4.1.2 声望影响来源

| 行为 | 声望变化 |
|------|----------|
| 完成派系任务 | +1（任务难度加成） |
| 护送派系成员 | +1 |
| 在派系控制区成功交易 | +0.2（累积） |
| 破坏派系货物 | -1 |
| 在派系领地内使用武力（非自卫） | -1 |
| 携带武器进入钢铁脊梁检查站 | -1（可提前说明避免） |
| 向净化派透露拾荒者信息 | 拾荒者-2，灰烬之子+1 |
| 提供稀缺物资（医疗/食物） | 根据派系需求+1~+2 |

#### 4.1.3 派系任务线

每局游戏，每个派系提供3条任务链，每条任务链包含3个任务节点：

```
任务链示例：拾荒者联盟 -「设备寻回」
  节点1：戈尔委托寻找一台旧世界医疗器械
    → 前往骨骸盆地某废墟
    → 触发：搜索事件（需要机械师）
    → 奖励：300瓶盖 + 拾荒者声望+1
    → 附加：发现旧世界日记残页（收藏品）

  节点2：戈尔告诉你医疗器械的卖家是谁
    → 护送物品前往钢铁脊梁边界
    → 触发：路线遭遇（可能被灰烬之子截获）
    → 选择：交给拾荒者 or 交给灰烬之子（影响双方声望）
    → 奖励：取决于选择

  节点3：戈尔分享一个秘密
    → 告诉你拾荒者变异免疫的真正原因
    → 解锁：零点碎片线索
    → 奖励：派系专属改装件图纸
```

### 4.2 货物与贸易系统（升级）

#### 4.2.1 货物分类

| 类型 | 货物 | 特点 | 时效 | 主要来源 |
|------|------|------|------|----------|
| 生存物资 | 食物、净水 | 高需求，不可缺 | 7天后质量下降（售价-20%） | 净土农联 |
| 医疗物资 | 药品、医疗器具 | 中等需求 | 无 | 灰烬之子、贸易站 |
| 工业零件 | 弹药、燃油桶、废金属、化学品 | 高需求 | 无 | 拾荒者联盟、废墟 |
| 高端货物 | 电子元件、奢侈品、织物 | 低需求高利润 | 无 | 军事基地、废墟 |
| 特殊货物 | 旧世界硬币、旧世界记录 | 极稀有 | 无 | 废墟探索、事件 |

#### 4.2.2 货物时效机制

- **保质期**：食物/净水的「质量值」每经过一次旅行下降
- **质量等级**：完好(100%) → 良好(75%) → 勉强(50%) → 腐败(无法出售，强制丢弃)
- **低温储存**：装有「冷藏模块」的改装件可减缓腐败速度
- **策略意义**：玩家需要在「高利润但时效短」和「低利润但稳定」之间做取舍

#### 4.2.3 派系需求波动

- 每隔3个旅行回合，各派系控制区的货物需求会更新
- 稀有事件（如瘟疫流行→医疗物资涨价3倍，蝗灾→食物涨价2倍）
- 派系战争会导致某些路线上的城镇货物价格剧烈波动
- **策略意义**：关注派系动态，在涨价前囤货，低价时卖出

### 4.3 乘员系统（升级）

#### 4.3.1 乘员职业

| 职业 | 名称 | 核心能力 | 升级效果 |
|------|------|----------|----------|
| 司机 | 领航员 | 降低油耗+5%/级，提升车速 | 油耗持续降低，路线规划更灵活 |
| 机械师 | 修理师 | 提升修车效率25%/级 | 车辆故障事件损失降低，能修理更多东西 |
| 保镖 | 护卫 | 战斗加成+12/级，降低被劫概率 | 战斗胜率明显提升 |
| **医护员**（新增） | 随队医生 | 每场旅行恢复乘员少量HP，防止食物腐败加速 | 减少减员，增强续航 |
| **向导**（新增） | 废土向导 | 降低遭遇强盗概率，路线信息更准确（显示预计事件数） | 更安全的长途旅行 |

#### 4.3.2 乘员羁绊系统

**羁绊条件**：
- 两个乘员同时参与同一旅行回合
- 完成特定任务（如同时参与一场战斗）
- 在酒馆休息时选择「团队活动」（消耗额外费用）

**羁绊效果**：
| 羁绊等级 | 效果 |
|----------|------|
| 熟悉（3次共同旅行） | 组合加成+10% |
| 信任（5次+共同任务） | 战斗时互相掩护——一方受伤另一方立刻援护 |
| 挚友（完成特定支线） | 解锁特殊组合技能（如「双保险」：保镖+向导联手，零伤害击退强盗） |

**背叛机制**：
- 乘员理智长期低于20%
- 连续不喂食（货舱食物为0超过3个旅行回合）
- 背叛概率触发事件：乘员会尝试偷取货物逃跑

#### 4.3.3 乘员个人剧情

每位非主角乘员有一条个人剧情线（共5个节点），完成全部节点解锁：
- 该乘员的「核心记忆」收藏品
- 永久专属被动技能
- 乘员专属结局触发条件

```
医护员剧情示例：「沉默的双手」
  节点1：在某次事件中，一位医护员使用了极其专业的急救技术
    → 询问她的过去：她曾是灰烬之子的医生，因某种原因离开
  节点2：她发现玩家经常在拾荒者联盟地盘活动
    → 选择：尊重她的隐私 or 追问（影响羁绊）
  节点3：灰烬之子派人来找她
    → 选择：帮助她躲避 or 交人（影响声望）
  节点4：她在废土上找到了她的家人
    → 她选择离开你的队伍（但留下了一本她写的医疗手册——物品效果）
  节点5：结局：如果她的羁绊等级达到「挚友」，她会回来参加你的结局
```

### 4.4 探索与路线系统（升级）

#### 4.4.1 路线信息面板

```
┌──────────────────────────────────────────────┐
│ 🛤️ 路线：铁锈镇 → 丰收镇                      │
├──────────────────────────────────────────────┤
│ 📏 距离: 120km    ⛽ 油耗: 45单位              │
│ ⚠️ 危险度: 35%    ☢️ 辐射: 无                 │
│ 🗺️ 地形: 锈蚀平原 → 再生绿洲                  │
│ 👥 势力: 拾荒者联盟 → 净土农联                │
├──────────────────────────────────────────────┤
│ ⚠️ 预计遭遇次数: ~1.8次                       │
│   ├ 强盗伏击概率: 12%（低）                   │
│   ├ 废弃车辆概率: 6%                         │
│   └ 幸存者概率: 5%                            │
├──────────────────────────────────────────────┤
│ 💡 锈轮商会情报：这条路线近期有钢铁脊梁巡逻队    │
│    通行更安全，但可能被盘问武器                 │
├──────────────────────────────────────────────┤
│        [ 🛒 出发！ ]                          │
└──────────────────────────────────────────────┘
```

#### 4.4.2 隐藏路线

通过特定事件或完成任务解锁隐藏路线：
- 完成钢铁脊梁探索任务 → 解锁「军事捷径」（距离缩短40%，但可能遇到旧军事设施）
- 完成拾荒者联盟探索任务 → 解锁「回收通道」（穿过废弃工厂区，有探索奖励但危险）
- 购买锈轮商会情报 → 获得某条路线的完整信息（包括隐藏捷径）

#### 4.4.3 旅行遭遇事件

**普通事件**（沿用v2.0系统，增加派系上下文）：
- 强盗伏击：根据所在地区不同，强盗的台词和装备会变化
- 废弃车辆搜索：根据城镇类型，搜出的货物有差异
- 幸存者：根据时间段不同，可能是难民、商人或逃亡者

**派系事件**（新增）：
- 派系检查站：钢铁脊梁检查武器，灰烬之子检查「变异迹象」
- 派系冲突：两个派系在路线上交战，绕行需要额外油耗
- 派系商队：跟随派系商队前进更安全，但需要缴纳「保护费」

**特殊事件**（与零点协议关联）：
- 零点碎片发现：在探索事件中有几率发现零点碎片
- 零点追寻者相遇：获得关于下一个碎片位置的线索
- 零点设施入口：收集7块碎片后解锁——进入后触发真结局

### 4.5 零点协议系统（终极目标）

#### 4.5.1 碎片收集

在游戏中，共有7块「零点碎片」，每块碎片包含一部分关于大寂静真相的信息。碎片通过以下方式获得：
1. 完成零点追寻者的探索任务
2. 在废墟探索事件中发现
3. 完成特定派系任务链的终章
4. 在某些特定角色的个人剧情中触发
5. 在特定路线的隐藏探索事件中找到

#### 4.5.2 碎片线索系统

每获得一块碎片，解锁一段信息片段，同时获得下一个碎片的大致位置线索：

```
碎片#1：「系统日志残页」
  内容：2077年11月14日 03:27:11
        「自动防御系统检测到[数据损坏]
         目标清单生成中...
         [警告] 目标清单包含[数据损坏]个条目
         授权码：零点
         执行确认：[已确认]」
  线索：骨骸盆地的某个旧军事设施可能有关联记录

碎片#2：「诺克斯动力内部备忘录」
  内容：诺克斯动力公司为[已涂黑]项目设计的
        原型能源系统。设计者备注：
        「当它启动的那一刻，
         我们就知道我们打开了潘多拉的盒子。
         但我们停不下来了。」
  线索：拾荒者联盟的「残骸王」戈尔可能知道更多

[...以此类推...]
```

#### 4.5.3 真结局：零点

收集全部7块碎片后，在任意派系声望达到「友善+」以上时，零点追寻者会联系你，邀请你一同前往「零点」——地图上唯一一座仍未被任何派系发现的零点设施入口。

真结局流程：
1. 跟随追寻者前往设施入口
2. 穿越一系列与之前碎片信息呼应的环境
3. 最终房间：面对零点博士的AI意识（或某种实体）
4. 选择：**启动系统** / **摧毁系统** / **带走数据离开**（三条真结局分支）
5. 根据你的派系关系和选择，获得对应的真结局独白

## 六、叙事设计

### 6.1 叙事节奏

```
回合 1-5：   【废土新人】→ 熟悉系统，建立基本策略
回合 6-15：  【派系接触】→ 各派系主动接触，提供任务
回合 16-25： 【深入抉择】→ 派系任务链推进，需要做出有代价的选择
回合 26-35： 【真相浮现】→ 零点碎片开始出现，线索汇聚
回合 36+：    【终局逼近】→ 所有派系线汇聚，结局条件逐渐明确
```

### 6.2 对话树设计

重要NPC对话采用分支对话树：

```
NPC：戈尔·芬顿（拾荒者联盟首领）
├── [尊重]「我来是想做一笔买卖。」
│   └── 戈尔：「买卖？好，我听听。」
│       ├── [直接]「听说你们在找某台医疗设备。」
│       │   └── 戈尔：「消息灵通。你想要什么？」（任务开启）
│       └── [试探]「我四处跑，能帮人找东西。」
│           └── 戈尔：「四处跑 eh...你知道拾荒者的规矩吗？」
│               └── 触发：拾荒者联盟背景介绍
│
├── [好奇]「残骸王座里面是什么样？」
│   └── 戈尔沉默片刻。「你来过就知道了。」
│       └── 戈尔：「不是所有人都欢迎好奇的客人。」（好感度微降）
│
└── [冒犯]「变异者真的存在吗？」
    └── 戈尔眼神变冷。「你想问什么？」
        ├── [解释]「只是听说，没别的意思。」
        │   └── 戈尔：「听说...废土上听说的事太多了。」（态度不变）
        └── [坚持]「灰烬之子说变异者是威胁。」
            └── 戈尔：「那你更应该知道——」
                「威胁的从来不是变异者，是那些把我们当威胁的人。」
                （解锁隐藏对话，获得额外背景信息）
```

### 6.3 叙事事件库

| 事件类型 | 触发条件 | 叙事内容 |
|----------|----------|----------|
| 派系首次接触 | 首次到达派系控制区城镇 | 该派系在你到达时的第一反应 |
| 派系冲突见证 | 在两派交界地带旅行 | 目击两派摩擦，有介入/绕行/无视三个选项 |
| 城镇危机 | 特定回合数+随机概率 | 城镇遭遇危机（辐射尘暴/强盗围城/瘟疫），需要帮助 |
| 零点碎片发现 | 持有碎片时探索废墟 | 与已有碎片信息呼应的环境叙事 |
| 派系结局倒计时 | 结局条件接近时 | 各派系对你的最后态度表述 |

## 七、技术架构

### 7.1 程序化生成

**地图生成**：
- 使用最小生成树算法确保地图连通性
- 派系控制区根据派系影响力动态分配（存在随机波动）
- 危险度、辐射、货物价格根据地形和派系计算
- 每次游戏生成不同地图，保证重玩性

**派系任务动态生成**：
- 基础任务模板库（每派系约15个基础任务）
- 任务难度根据游戏进度（回合数）动态调整
- 任务结果影响派系关系和后续任务链走向

**城镇人物生成**：
- 核心人物（派系领袖、关键NPC）：每局固定生成，配有完整对话树
- 地方人物：模板生成，包含随机化的背景细节和对话变体
- 城镇氛围词：根据派系+地形+游戏阶段生成城市特征描述

### 7.2 数据结构

```
GameState {
  // 世界
  map: { towns[], routes[], factions[] }

  // 玩家状态
  player: {
    money, oldWorldCurrency, reputation: { [factionId]: score },
    vehicle: Vehicle,
    crew: CrewMember[],
    cargo: { [goodId]: { amount, quality } },
    reputation: { [factionId]: number },
    inventory: Item[]
  }

  // 进度
  currentTownId: number,
  turn: number,
  questProgress: { [questId]: stage },
  collectibles: { [collectibleId]: boolean },
  zeroFragments: ZeroFragment[]

  // 存档
  saveSlot: 0 | 1 | 2 | 3
  gameMode: 'main' | 'freeplay' | 'achievements'
  endingConditions: { [endingId]: boolean }
}
```

### 7.3 性能目标

- 启动时间：< 3秒
- 每回合响应：< 100ms
- 存档大小：< 500KB
- 帧率：稳定60fps（Canvas渲染）

## 八、难度设计

### 8.1 难度级别

| 难度 | 燃油消耗 | 事件概率 | 货物价格 | 派系敌意 |
|------|----------|----------|----------|----------|
| 轻松 | ×0.7 | ×0.7 | 稳定 | 降低敌意速度 |
| 普通 | ×1.0 | ×1.0 | 正常波动 | 正常 |
| 困难 | ×1.3 | ×1.2 | 大幅波动 | 敌意上升更快 |
| 废土老兵 | ×1.5 | ×1.5 | 剧烈波动 | 初始中立，所有派系敌意+1 |

### 8.2 平衡关键点

- **燃油经济**：油耗太高→玩家被迫频繁加油→失去探索动力
- **货物利润**：利润太低→交易系统失去意义；利润太高→游戏失去挑战
- **派系惩罚**：声望降低惩罚太重→玩家不敢做任何选择；太轻→派系系统无意义
- **战斗难度**：战斗太难→玩家只能规避；太简单→战斗失去紧张感
